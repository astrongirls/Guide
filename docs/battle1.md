# 戦闘概要

# 戦闘とは？

「アストロアンドガールズ」における戦闘とは、宇宙空間で艦隊と艦隊が激突し、勝敗を競うことを言います。

<img src="_images/10.png" width="50%"><img src="_images/34.png" width="50%">

戦闘をするためには艦隊と艦長が必要です。

戦闘で勝利するためには高い性能を持った艦隊と武器が必要です。

戦闘で勝利するためには高い能力を保有する英雄が必要です。

英雄が保有する戦闘スキルは勝敗を大きく左右します。

戦闘に勝利するためには相手艦隊をすべて撃破する必要があります。

決められたステージ（ターン）内で相手の艦隊をすべて撃破できない場合、相手の防御システムが作動し自分の艦隊は退却することになります。

戦闘に参加した艦隊が敵軍の攻撃により被害を受け、その中でも大破した艦船は、新たに生産し補充しなければいけません。

戦闘に勝利しても、自分の艦隊が被害を受けることもあります。

# 戦争

コスメントの採掘が可能な3つの地域で順次戦争が勃発します。

個人間の戦闘結果を点数で集計し、勝利陣営を決定します。

個人間の勝利に対する報酬と共に最終勝利した陣営は、コスメントを採掘する権限が与えられます。

每日23:00 - 翌22:00までの23時間の間、コスメントをめぐって勃発する紛争です。

各陣営の貢献度を集計して、勝利陣営が決定されます。

20:00-22:00の間は、得られる貢献度がUPします。

※22:00-23:00の間は、次の戦争の準備をしています。

# 特殊任務
シナリオに沿って宇宙海賊を追跡し撃退します。

勝利すると報酬アイテムを獲得することができます。

報酬アイテムにはトロンを製造できる部品が含まれます。

# 撃退任務

撃退任務をしていると、一定確率で外界艦隊に遭遇します。

攻撃するとゴールドと鉱物を獲得できます。

外界艦隊の撃退に成功すると、外界技術と関連した報酬を受け取ることができます。

# 討伐
シナリオに沿って悪党の本拠地に攻撃をしに行く任務です。

ステージに勝利するとゴールドと鉱物、DNAコア、特定のドロップアイテムを受け取ることができます。

星の獲得個数によって追加で報酬を受けることができます。

# リーグ

リーグは50レベル以上になると楽しめるコンテンツです。

同水準の国力の国家をマッチングさせ、お互いに競争させます。

防御フォーメーションを公開し、戦闘結果を細分化することで、より効率的な戦闘のための戦略が可能となります。

毎ラウンドごとにコスメント資源を獲得することができ、上位に進むほど受け取れる報酬も多くなります。

リーグポイントを通じて様々アイテムと交換することもできます。



# 征伐
シナリオに沿って悪党の本拠地に攻撃する任務です。

ステージに勝利するとゴールドと鉱物、DNAコア、特定のドロップボックスを受け取ることができます。

星の獲得個数によって追加で報酬を受けることができます。


# 討伐

後方を攪乱する悪党を討伐するための任務です。

討伐任務は国家レベルが20レベルに達すると開始できます。

# 勝利のための戦略

兵器と艦体はティアが高くなるほど、性能が優れています。

高い性能を持つ兵器や艦隊は戦闘で勝つうえで必須条件です。

英雄が持っている様々な職業やスキルをうまく活用する必要があります。

職業に合った艦体や兵器を選択しなければなりません。

スキルに合った兵器を選択することでスキルの威力を上げることができます。

新しい兵器や艦体を作ることができない場合、研究に力を入れる必要があります。

基礎研究には兵器や艦体の性能を上げる研究があります。

戦闘で勝利するためには相性の概念を十分に理解する必要があります。

兵器と艦体には相性が存在し、特定の兵器に相性の良い艦体を合わせることで強くすることができます。

スキルにはスキル同士の相性が存在します。特定の攻撃スキルには相性の良い防御スキルで対応すると効率が上昇します。

特定のフォーメーションの相手に戦闘で勝利するためには、相手のフォーメーションとは対極の兵器を装着したり、相手に合わせたフォーメーションを選択する必要があります。

最も重要なのは相手との相性です。自分が必ず勝利したい相手であれば、相手の英雄と艦隊の情報を把握することが何よりも重要になってきます。

艦隊の艦船の数が多い場合、強力な敵軍のターゲットになることがあります。

逆に艦隊の艦船の数が十分でない場合、強力な攻撃をすることができません。
