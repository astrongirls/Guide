# LD交換所
<img src="_images/43.png" width="100%">

・LD購入

LDチップとクリスタルを使用して、スキルLDを入手することができます。

このリストは每日更新されますが、クリスタルを使用することですぐに更新することもできます。

・訓練LD交換

一般訓練LD10個 → 高級訓練LD1個

高級訓練LD1個 → 一般訓練LD5個

に交換することができます。
